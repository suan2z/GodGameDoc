## 🎯 핵심 컨셉

> "친구와 함께, 카드를 써서, 적 타워를 부숴라!"

친구와 함께 전략 카드를 내려 적의 타워를 무너뜨리는 **실시간 코옵 타워 디펜스 게임**

| 컨셉 | 설명 |
|------|------|
| 🏰 **타워 디펜스** | 적의 공격을 막으면서 아군 타워를 지키는 전략 방어 게임 |
| 🤝 **코옵 플레이** | 친구와 함께 협력해서 싸우는 멀티플레이 중심 경험 |
| 🃏 **카드 시스템** | 클래시 로얄처럼 카드를 골라 유닛·마법을 전장에 배치 |
| ⚡ **실시간 전략** | 빠른 판단과 팀워크가 승패를 가르는 실시간 배틀 |

---

## 🖼️ 참고 화면 — 클래시 로얄 스타일

게임은 클래시 로얄과 유사한 전투 화면 구조를 채택합니다.

- 상단에 **적 타워 HP** 표시
- 하단에 **카드 패 및 엘릭서(마나) 바** 위치
- 세로형 전투 화면 기본 채택
- 코옵 특성상 화면 분할 또는 역할 분담 UI가 추가될 수 있음

---

## ⚔️ 게임플레이

### 기본 전투 흐름

1. **2명의 플레이어**가 코옵으로 전투에 입장한다
2. 전투는 **타워 디펜스** 규칙으로 진행된다
3. **웨이브마다 적들이 등장**하며 아군 진영을 향해 진격한다
4. 적들은 **상단의 3개 다리**를 기준으로 **3개의 길**에서 동시에 몰려온다
5. **마지막 웨이브까지 적들을 모두 처치**하면 스테이지 클리어
6. **아군 타워가 부서지면 패배**

### 기본 스펙 (초안)

| 항목 | 내용 | 상태 |
|------|------|------|
| 전투 인원 | 2v2 (또는 1v1) | 검토 중 |
| 게임 시간 | 3분 내외 단판 배틀 | 검토 중 |
| 뷰 방식 | 클래시 로얄 스타일 세로형 탑뷰 | **확정** |
| 카드 수 | 덱 구성 카드 수 | 논의 필요 |
| 플랫폼 | 미정 | 논의 필요 |

## 🤝 코옵 시스템

일반 타워 디펜스와 달리, 갓겜은 **협력**이 승리의 핵심입니다.

- 파트너와 **실시간으로 소통**하며 카드를 배치
- 각자의 **역할 분담** (공격형 / 수비형) 가능성 검토 중
- 팀원과 카드 또는 자원을 **공유하는 메커닉** 도입 검토

> ⚠️ 코옵 메커닉 세부 내용(역할 분담, 공유 시스템, 통신 방식 등)은 추후 별도 문서로 정의 예정

## 🎮 타겟 & 방향성

### 타겟 유저

- 클래시 로얄, 클래시 오브 클랜 등 **캐주얼 전략 게임** 유저
- 친구와 함께 가볍게 즐기는 **코옵 게임**을 선호하는 플레이어
- 짧은 플레이 시간 안에 **전략적 재미**를 원하는 유저

### 레퍼런스 게임

| 게임 | 참고 요소 |
|------|-----------|
| 클래시 로얄 | 전투 화면 구조, 카드 시스템, 엘릭서 메커닉 |
| 클래시 오브 클랜 | 타워 배치, 유닛 종류, 아트 스타일 방향 |
