게임 페이즈 상세

Phase 1 — 드래프트 (웨이브 시작 전 준비 시간)

  • 각 웨이브 시작 전 15초 드래프트 타임 제공
  • 플레이어는 골드를 소비해 유닛을 랜덤 소환 (가챠 방식)
  • 소환된 유닛은 즉시 배치 가능 / 판매 가능
  • 웨이브 1~10: 기본 소환만 가능
  • 웨이브 11~30: 고급 소환 해금
  • 웨이브 31~50: 전설 소환 해금

Phase 2 — 배치

  • 드래프트 이후 10초 배치 타임
  • 자신의 배치 구역(맵 절반)에 유닛 자유 배치
  • 상대 배치 영역은 반투명으로 실시간 확인 가능

Phase 3 — 전투 (자동 전투)

  • 배치 완료 후 자동 전투 진행
  • 플레이어는 전투 중 스킬 액티브 가능 (웨이브 당 1회)
  • 웨이브 종료 시 골드 획득

매칭 시스템

매칭 알고리즘

  • 같은 난이도 선택자끼리만 매칭
  • 매칭 우선순위: ① 같은 난이도 ② 비슷한 레벨 범위 (±10 레벨)
  • 매칭 대기 30초 초과 시: 레벨 범위 ±20으로 확장
  • 매칭 대기 60초 초과 시: AI 봇 매칭으로 대체 (봇 표시 없음, 퀄리티 동일하게 설계)

  • 프로토콜: WebSocket 기반 실시간 통신

  • 패킷 위변조 방지: 서버 사이드 밸리데이션 필수
  • 탈주/연결 끊김: 30초 내 재연결 허용, 초과 시 AI 대체

  • 유저 설정, 게임 팁 등 경량 데이터는 로컬 저장
  • 결제 및 게임 데이터는 서버 우선 (서버 권위 방식)
