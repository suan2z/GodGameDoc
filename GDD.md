## 1. 게임 개요

### 1.1 기본 정보

| 항목 | 내용 |
|------|------|
| **게임명** | 갓겜 (GodGame) |
| **장르** | 캐주얼 전략 랜덤 디펜스 |
| **플랫폼** | iOS / Android |
| **플레이 인원** | 2인 코옵 (온라인 매칭) |
| **타겟 유저** | 15~35세, 가볍게 즐기는 캐주얼 게이머 |
| **등급** | 전체 이용가 (심의 예정) |
| **핵심 키워드** | 랜덤, 코옵, 운빨, 디펜스, 짧은 세션 |

### 1.2 엘리베이터 피치

> "운빨로 뽑은 유닛으로 낯선 유저와 함께 몰려오는 적을 막는 2인 코옵 디펜스.  
> 이기면 운빨, 지면 팀운 — 그래도 계속하게 되는 게임."

### 1.3 게임 목표

- **단기 (MVP)**: 핵심 디펜스 루프 + 2인 코옵 매칭 완성
- **장기**: 글로벌 캐주얼 모바일 시장 공략, 시즌제 운영

## 3. 게임플레이 메카닉

### 3.1 전체 게임 플로우

```
[로비] → [난이도 선택] → [매칭 대기] → [유닛 드래프트 Phase]
  → [배치 Phase] → [웨이브 전투] → [반복 (다음 웨이브)]
  → [50웨이브 클리어 or 실패] → [결과 화면] → [보상 지급] → [로비]
```

### 3.2 게임 페이즈 상세

#### Phase 1 — 드래프트 (웨이브 시작 전 준비 시간)

- 각 웨이브 시작 전 **15초** 드래프트 타임 제공
- 플레이어는 **골드**를 소비해 유닛을 **랜덤 소환** (가챠 방식)
- 소환된 유닛은 즉시 배치 가능 / 판매 가능
- 웨이브 1~10: 기본 소환만 가능
- 웨이브 11~30: 고급 소환 해금
- 웨이브 31~50: 전설 소환 해금

#### Phase 2 — 배치

- 드래프트 이후 **10초** 배치 타임
- 자신의 배치 구역(맵 절반)에 유닛 자유 배치
- 상대 배치 영역은 반투명으로 실시간 확인 가능

#### Phase 3 — 전투 (자동 전투)

- 배치 완료 후 자동 전투 진행
- 플레이어는 전투 중 **스킬 액티브** 가능 (웨이브 당 1회)
- 웨이브 종료 시 골드 획득

### 3.3 골드 시스템

#### 골드 획득

| 획득 방법 | 골드량 |
|-----------|--------|
| 웨이브 클리어 | 10 + (웨이브 번호 × 2) |
| 적 처치 보너스 | 적 1마리당 1골드 |
| 콤보 보너스 (연속 클리어) | 5골드 추가 |
| 유닛 판매 | 구매가의 60% 반환 |

#### 소환 비용 및 확률

| 소환 종류 | 비용 | 등급 확률 |
|-----------|------|-----------|
| 기본 소환 | 3골드 | 일반 70% / 희귀 25% / 영웅 5% |
| 고급 소환 | 8골드 | 일반 20% / 희귀 50% / 영웅 25% / 전설 5% |
| 전설 소환 | 20골드 | 영웅 40% / 전설 55% / 신화 5% |

> ⚠️ **밸런스 메모**: 웨이브 31+ 모든 소환 비용 +2골드 자동 적용 (골드 인플레이션 방지)

## 4. 메인 스테이지 룰

### 4.1 기본 구조

- **인원**: 2인 1팀 온라인 코옵
- **목표**: 50웨이브를 끝까지 막아내기
- **실패 조건**: 팀 공유 체력(라이프 포인트) 0 도달

### 4.2 난이도 3종

| 난이도 | 이름 | 웨이브 | 적 특성 | 권장 레벨 | 보상 |
|--------|------|--------|---------|-----------|------|
| 쉬움 | 🌱 일반 | 50 | 기본 속도, 기본 HP | 제한 없음 | 기본 보상 |
| 보통 | 🔥 지옥 | 50 | +30% 속도, +50% HP | Lv.10+ | 1.5배 보상 |
| 어려움 | 💀 신의 시련 | 50 | +70% 속도, 특수 적 등장, 보스 강화 | Lv.25+ | 3배 보상 |

> 매칭 규칙: 같은 난이도를 선택한 유저끼리 자동 매칭

### 4.3 라이프 포인트 (LP) 시스템

- 팀 공유 라이프: **20 LP**
- 적 1마리 통과 시: **-1 LP** (일반) / **-2 LP** (정예) / **-5 LP** (보스)
- LP 회복: 특정 웨이브 클리어 보너스 (웨이브 10, 25, 40)에서 **+2 LP** 지급

### 4.4 웨이브 구조 (50웨이브)

| 구간 | 웨이브 | 특징 |
|------|--------|------|
| 초반 | 1~10 | 기본 적, 난이도 입문 구간 |
| 중반 | 11~25 | 정예 적 등장, 적 수 증가 |
| 후반 | 26~40 | 빠른 적 / 탱커 혼합, 전략 필요 |
| 위기 | 41~49 | 보스 미니언 + 정예 혼합 |
| 최종 | 50 | **보스 웨이브** — 대형 보스 1마리 + 호위대 |

### 4.5 코옵 인터랙션

- **유닛 공유 불가** — 각자 독립 배치 구역
- **시너지 연동 가능** — 상대 유닛 등급 확인하여 보완 배치 전략
- **이모티콘 소통** — 텍스트 채팅 없이 빠른 이모티콘 6종으로 소통
- **서포트 스킬** — 일부 유닛이 상대방 구역에도 버프 제공

## 5. 유닛 시스템

### 5.1 유닛 등급

| 등급 | 이름 | 색상 | 기본 스탯 배율 | 스킬 수 |
|------|------|------|----------------|---------|
| 1등급 | 일반 | 회색 | × 1.0 | 0 |
| 2등급 | 희귀 | 파랑 | × 1.8 | 1 (패시브) |
| 3등급 | 영웅 | 보라 | × 3.5 | 1 (액티브) |
| 4등급 | 전설 | 금색 | × 7.0 | 2 |
| 5등급 | 신화 | 무지개 | × 15.0 | 3 |

### 5.2 유닛 타입

| 타입 | 역할 | 공격 방식 | 배치 위치 |
|------|------|-----------|-----------|
| ⚔️ 전사 | 탱커 | 근접 단일 | 전선 |
| 🏹 궁수 | 원딜 | 원거리 단일 | 후방 |
| 🔮 마법사 | 광역딜 | 원거리 범위 | 중앙 |
| 💚 힐러 | 지원 | 아군 회복 | 후방 |
| 🗡️ 암살자 | 고딜 | 근접 단일 (빠름) | 측면 |
| 💣 대포 | 공성 | 범위 느림 | 고정 포대 |

### 5.3 유닛 합성 시스템

**동일 유닛 3개 합성 → 다음 등급 1개 승급**

```
일반 × 3 → 희귀 1개 → 영웅 1개 → 전설 1개 → 신화 1개
```

> 무료 플레이어도 합성으로 고등급 유닛에 도달 가능하도록 설계

### 5.4 시너지 시스템

같은 **속성**의 유닛 N개 배치 시 팀 전체 버프 적용

| 속성 | 2개 시너지 | 4개 시너지 | 6개 시너지 |
|------|-----------|-----------|-----------|
| 🔥 불꽃 | 공격력 +10% | 공격력 +25% | 범위 공격 발동 |
| ❄️ 얼음 | 적 둔화 5% | 적 둔화 15% | 적 빙결 확률 |
| ⚡ 번개 | 크리티컬 +5% | 크리티컬 +15% | 연쇄 번개 발동 |
| 🌑 어둠 | HP 흡수 3% | HP 흡수 10% | 흡혈 광역화 |
| ✨ 빛 | 아군 회복 5 | 아군 회복 15 | 부활 1회 |

## 6. 적 시스템

### 6.1 적 종류

| 분류 | 이름 | 특성 | LP 피해 |
|------|------|------|---------|
| 기본 | 슬라임 / 고블린 | 기본 스탯 | -1 |
| 빠름 | 늑대 / 스프린터 | 이동속도 2배 | -1 |
| 탱커 | 거인 / 실드몹 | HP 3배, 느림 | -2 |
| 정예 | 엘리트 몬스터 | 스탯 2배 + 스킬 보유 | -2 |
| 보스 미니언 | 보스 수행원 | 보스 웨이브 전 등장 | -3 |
| 보스 | 대형 보스 (웨이브 50) | 엄청난 HP + 패턴 3종 | -5 |

### 6.2 웨이브 당 적 수

| 웨이브 | 적 수 |
|--------|-------|
| 웨이브 1 | 10마리 |
| 웨이브 10 | 30마리 |
| 웨이브 25 | 50마리 |
| 웨이브 40 | 70마리 |
| 웨이브 50 | 보스 1마리 + 수행원 30마리 |

### 6.3 보스 패턴 (웨이브 50)

| 패턴 | 발동 조건 | 효과 |
|------|-----------|------|
| 패턴 1: 돌진 | HP 100% | 직선 돌진, 전선 유닛 넉백 |
| 패턴 2: 광역 공격 | HP 60% | 배치 구역 전체 데미지 |
| 패턴 3: 증원 소환 | HP 30% | 정예 몬스터 10마리 소환 |

## 7. 맵 & 레이아웃

### 7.1 맵 구조

```
[적 출현 지점]
       ↓
+--------------------+
|  경로 (Path)       |  ← 구불구불한 경로, 유닛은 경로 밖 배치
|  플레이어 A 구역   |  ← 맵 좌측 절반
|  플레이어 B 구역   |  ← 맵 우측 절반
+--------------------+
       ↓
[라이프 포인트 기지]
```

- 뷰 방식: **클래시 로얄 스타일 세로형 탑뷰** (확정)
- 적들은 **3개의 길**에서 동시에 몰려오는 구조 (상단 기준 3개 다리)

### 7.2 맵 타일 규격

- 그리드 기반: **8 × 12 타일** (각 플레이어 절반)
- 배치 가능 타일: 경로 제외 나머지 (약 70%)
- 경로 타일: 배치 불가, 시각적으로 구분

### 7.3 맵 테마 (스테이지별 변형)

| 테마 | 특징 |
|------|------|
| 🌿 초원 | 기본 맵, 직선 경로 |
| 🏚️ 던전 | 구불구불한 경로, 시야 제한 |
| ❄️ 설원 | 유닛 이동속도 감소 디버프 구역 포함 |
| 🌋 화산 | 주기적 화염 타일 발동 |
| 🏛️ 신전 | 보스 강화 존 존재 |

## 8. 매칭 시스템

### 8.1 매칭 알고리즘

- **같은 난이도 선택자끼리만 매칭**
- 매칭 우선순위: ① 같은 난이도 → ② 비슷한 레벨 범위 (±10 레벨)
- 매칭 대기 30초 초과 시: 레벨 범위 ±20으로 확장
- 매칭 대기 60초 초과 시: AI 봇 매칭으로 대체 (봇 표시 없음, 퀄리티 동일하게 설계)

### 8.2 매칭 화면 UX 흐름

```
[난이도 선택 버튼 3개]
    ↓ 선택
[매칭 대기 화면]
  - 상대방 닉네임 + 레벨 표시
  - 예상 대기 시간 카운트
  - 매칭 취소 버튼
    ↓ 매칭 성공
[게임 시작 카운트다운 3초]
```

### 8.3 탈주 패널티

- 웨이브 10 이전 탈주: 경험치 0, 골드 0, 탈주 기록 누적
- 3회 누적 탈주: **10분 매칭 제한**
- 웨이브 10 이후 탈주: AI 자동 대체 (상대방 불이익 최소화)

## 9. 성장 & 메타 시스템

### 9.1 플레이어 레벨

- 경험치 획득: 스테이지 클리어, 일일 퀘스트, 업적
- 레벨 보상: 소환권, 골드, 코스튬 조각

### 9.2 유닛 영구 강화 (컬렉션)

- 동일 유닛 다수 획득 시 **컬렉션 포인트** 누적
- 컬렉션 포인트로 해당 유닛 영구 스탯 강화 (최대 **+30%**)
- 이 강화는 게임 내 전투에서 항상 적용

### 9.3 업적 & 칭호

| 업적명 | 조건 | 보상 |
|--------|------|------|
| 첫 클리어 | 스테이지 최초 클리어 | 소환권 10장 |
| 50웨이브 완파 | 일반 난이도 클리어 | 타이틀 "디펜더" |
| 지옥에서 살아남다 | 지옥 난이도 클리어 | 타이틀 "지옥전사" |
| 신화를 소환하라 | 신화 등급 유닛 1회 획득 | 코스튬 조각 |
| 탈주 없는 50판 | 50판 탈주 없이 완주 | 전용 프레임 |

### 9.4 시즌 패스 트랙

## 11. 클라이언트 기술 사양

| 항목 | 내용 |
|------|------|

### 11.2 클라이언트-서버 통신

- 프로토콜: WebSocket 기반 실시간 통신

- 배치 정보는 드래프트 종료 시 **양쪽 동시 전송**
- 패킷 위변조 방지: 서버 사이드 밸리데이션 필수
- 탈주/연결 끊김: 30초 내 재연결 허용, 초과 시 AI 대체

### 11.3 로컬 데이터 관리

- 유저 설정, 게임 팁 등 경량 데이터는 로컬 저장
- 결제 및 게임 데이터는 서버 우선 (서버 권위 방식)

### 12.1 서버 구성

- **매칭**: 난이도별 매칭 요청 / 취소 / 성공 콜백
- **게임 진행**: 웨이브 시작, 배치 정보 전송, 결과 저장

- **성장**: 경험치/레벨 처리, 업적 달성, 컬렉션 업데이트
- **결제**: 영수증 서버 검증 (Google Play / App Store)

### 12.3 보안 및 Anti-Cheat

- 결제 검증: 영수증 서버 검증 (Google Play / App Store)

- 소환 추가 1회 (광고 시청, 1일 3회 제한)
- 일일 골드 2배 (광고 시청, 1일 1회)

> 광고 제거 상품: ₩3,300 (영구) — 게임 내 배너/전면 광고 제거 (보상형은 선택 유지)

- 개별 판매 ₩1,100~₩5,500 / 번들 ₩8,900

> 소환권으로 강한 유닛 확보 가능하나, 합성으로 무료 플레이어도 도달 가능하게 설계.  
> 광고 보상은 편의 제공 수준으로 제한 (게임 불균형 야기 금지).
